# Sales Feature (`features/sales/`)

## الغرض
شاشة عرض وتتبع المبيعات المكتملة مع ملخصات إحصائية.

## البنية
```
sales/
  presentation/
    pages/
      sales_page.dart    ← الصفحة الرئيسية: ملخصات + قائمة الفواتير
```

## المكونات الرئيسية
- **SalesPage**: تعرض ملخص مبيعات اليوم والإجمالي (إيرادات، أرباح، عدد فواتير) + قائمة الفواتير مع إمكانية عرض تفاصيل كل فاتورة.

## قواعد العمل
- جميع المبالغ `int` (بدون كسور).
- البيانات تُجلب من `DatabaseHelper` عبر: `getSalesSummary()`, `getTodaySales()`, `getAllSales()`, `getSaleItems()`.
- يمكن التبديل بين عرض مبيعات اليوم والكل.
- الضغط على فاتورة يعرض تفاصيل المنتجات في BottomSheet.
