# Theme Module Documentation

## File: app_theme.dart

### AppTheme Class
Contains application-wide theming configuration. All colors MUST be defined here — no hardcoded colors in UI files.

#### Color Constants:
- `primaryColor` — بنفسجي أنيق (0xFF673AB7)
- `secondaryColor` — بنفسجي فاتح (0xFFE1BEE7)
- `backgroundColor` — رصاصي فاتح للخلفية (0xFFF5F5F5)
- `surfaceColor` — أبيض للبطاقات
- `textPrimary` — أسود داكن للنصوص (0xFF212121)
- `textSecondary` — رصاصي للنصوص الثانوية (0xFF757575)
- `errorColor` — أحمر للأخطاء والحذف (0xFFD32F2F)
- `successColor` — أخضر للنجاح والأرباح (0xFF388E3C)
- `warningColor` — برتقالي للتحذيرات والتعليق (0xFFFF9800)
- `highlightColor` — أصفر فاتح للتحديد (0xFFFFF8E1)
- `searchFieldColor` — أزرق فاتح لحقل البحث (0xFFE3F2FD)
- `neutralColor` — رصاصي للعناصر المحايدة (0xFF9E9E9E)
- `neutralLightColor` — رصاصي فاتح للخلفيات (0xFFF5F5F5)

#### ThemeData:
- `lightTheme` — Full ThemeData with Tahoma font, RTL support
